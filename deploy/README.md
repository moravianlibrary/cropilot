## Deployment

Cropilot is officially available at [app.cropilot.cz](https://www.app.cropilot.cz). If you want to self-host, the app can be deployed using Docker Compose or Helm for Kubernetes deployment. The production setup includes the backend API, worker, Hatchet queue manager, MongoDB, and frontend services.

See the [Docker Deployment Guide](./deploy/docker/README.md) or [Kubernetes Deployment Guide](./deploy/kubernetes/README.md) for full setup instructions.
